---
description: Commit and deploy changes
---

Pokud ti zadám, abys nahrál/nasadil změny, udělej tyto kroky:
1. Commitni všechny změny do master branch
2. Pushni změny na remote branch
3. Kontroluj kdy dobehne github pages deploy (pres command "gh run watch") a potom informuj, ze stranky byly nasazeny
4. Vypis URL stranek do chatu jako odkaz
